    «Альт Сервер» —

    серверный дистрибутив с широкой функциональностью, позволяющий разворачивать и поддерживать корпоративную инфраструктуру предприятий, а также использовать разнообразное периферийное оборудование.

    Почему используют «Альт Сервер»

    Российское ПО, входит в реестр Минцифры (регистрационный номер ПО: 1541).
    Дистрибутив выпускается в нескольких вариантах исполнения: для x86 (64-битных), AArch64 (Huawei Kunpeng, ThunderX и другие), ppc64le (Power 8 и 9, OpenPower), e2kv3/e2kv4/e2kv5/e2kv6 («Эльбрус»).
    Содержит сетевые службы и сервисы для полноценного функционирования корпоративной инфраструктуры.
    Включает контроллеры доменов: ALT-домен (Samba-DC c поддержкой групповых политик для бесшовной миграции с Windows на Linux), FreeIPA (для централизованной аутентификации пользователей).
    Безопасные* настройки конфигурации.
    Для построения офисной инфраструктуры в рамках импортозамещения рекомендуется использовать вместе с «Альт Рабочая станция».
    Совместимость с экосистемой ПО отечественных производителей.
    Система управления сервером имеет более 100 модулей и позволяет работать через графический или веб-интерфейс.

    Сроки поддержки в части обновлений по безопасности (если иное не предусмотрено условиями поставки):

    • для дистрибутивов «Альт Сервер» 9 версий — 31 декабря 2023, но не ранее полугода с момента выпуска новой версии (10.0);
    • для дистрибутивов «Альт Сервер» 10 версий — 31 декабря 2024, но не ранее полугода с момента выпуска новой версии (11.0).

    «Альт Сервер» предоставляет полный набор служб
    для создания корпоративной инфраструктуры

    «Альт Cервер» предоставляет следующие виды контроллеров домена:

    Samba-DC

    (контроллер домена Active Directory)

    Рекомендуется для аутентификации рабочих станций под управлением Windows и Linux (гетерогенная сеть).

    FreeIPA

    (средство централизованной аутентификации пользователей)

    Компоненты для централизованного управления пользователями с поддержкой политик доступа и аудита.

    Для построения офисной инфраструктуры в рамках импортозамещения рекомендуется использовать вместе с продуктом «Альт Рабочая станция» как стабильное и надежное решение.
